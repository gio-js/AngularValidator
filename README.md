# ng5-validator

Angular 5 - Forms validation


## Getting started

`npm i --save ng5-validator`

Add following lines into your

**module:**

```typescript
import { ValidatorsModule } from './ng5-validator';
```

add ValidatorsModule to your module imports section<br/>
```typescript
imports: [ ValidatorsModule ]
```
